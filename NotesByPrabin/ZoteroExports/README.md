# 🔬 ZoteroExports

Raw academic reference exports from [Zotero](https://www.zotero.org/), imported into Obsidian via the [Zotero Integration plugin](https://github.com/mgmeyers/obsidian-zotero-integration).

## What Lives Here

- **Auto-generated reference notes** — created by the Zotero Integration plugin from your Zotero library
- **PDF annotations** — highlights and comments from PDFs read in Zotero, imported as Markdown
- **Citation metadata** — author, title, year, DOI, abstract pulled directly from Zotero
- **Raw highlights** — unprocessed extraction from papers and articles

## Workflow

```
Add paper to Zotero → Read & annotate in Zotero PDF viewer
        ↓
Run Zotero Integration import in Obsidian
        ↓
Raw note lands in ZoteroExports/
        ↓
Process: extract key insights, write synthesis
        ↓
Refined note moves to Bibliography/ or ATLAS/
```

## Important

> ⚠️ Files in this folder are **auto-generated** by the Zotero Integration plugin. Do not manually edit them — re-importing from Zotero will overwrite your changes. Process notes by copying insights to `Bibliography/` instead.

## About Zotero

[Zotero](https://www.zotero.org/) is a free, open-source reference manager for collecting, organizing, and citing research sources. Combined with the Obsidian Zotero Integration plugin, it creates a seamless academic reading and note-taking pipeline.
