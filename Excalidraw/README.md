# ✏️ Excalidraw

Visual thinking space. This folder stores all whiteboard diagrams, mind maps, system sketches, and visual notes created with the [Excalidraw plugin](https://github.com/zsviczian/obsidian-excalidraw-plugin) for Obsidian.

## What Lives Here

- **Concept diagrams** — visual representations of complex ideas or systems
- **Mind maps** — radial explorations of a topic
- **System sketches** — architecture diagrams, flowcharts, and process maps
- **Brainstorm canvases** — freeform visual exploration during ideation
- **Embedded visuals** — `.excalidraw` files that are embedded into Markdown notes

## File Format

Excalidraw files are saved as `.excalidraw` (JSON) or `.md` with embedded Excalidraw data. Both formats are handled natively by the Obsidian Excalidraw plugin.

## Embedding in Notes

A diagram stored here can be embedded in any note:

```markdown
![[Excalidraw/my-diagram.excalidraw]]
```

The diagram renders inline in Reading Mode.

## ExcaliBrain

The `excalibrain.md` in the vault root uses [ExcaliBrain](https://github.com/zsviczian/excalibrain) to generate a dynamic visual graph of all vault note relationships, powered by the Excalidraw engine.
