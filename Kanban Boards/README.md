# 📌 Kanban Boards

Visual task and project management boards powered by the [Obsidian Kanban plugin](https://github.com/mgmeyers/obsidian-kanban). Each board is a Markdown file rendered as a drag-and-drop Kanban.

## What Lives Here

- **Project boards** — task tracking for active efforts from `EFFORTS/`
- **Weekly task boards** — what needs to happen this week
- **Study boards** — tracking progress through courses or reading lists
- **Habit boards** — recurring tasks and routines

## Board Structure

Each Kanban board uses columns to represent task states:

```
Backlog → In Progress → Review → Done
```

Cards are plain Markdown checkboxes under each column header, making them fully readable as plain text even without the plugin.

## Integration with EFFORTS

Kanban boards are the **tactical layer** under each project note in `EFFORTS/`. The project note defines the *what* and *why*; the Kanban board tracks the *tasks* needed to get there.

> 📌 Tasks without a system stay as wishes. Kanban turns intentions into a visible, actionable workflow.
