# 📚 Bibliography

Reference notes, source summaries, and annotated citations. This folder is the vault's **academic and research layer** — every book, paper, article, or talk that has influenced thinking in this vault has an entry here.

## What Lives Here

- **Book notes** — summaries, key takeaways, and highlights from books
- **Paper notes** — summaries and annotations of academic papers
- **Source citations** — properly formatted references ready to link from other notes
- **Reading highlights** — important quotes and passages with context

## Relationship with Zotero

Academic references are primarily managed in [Zotero](https://www.zotero.org/) and exported into this vault via the [Zotero Integration plugin](https://github.com/mgmeyers/obsidian-zotero-integration). Raw Zotero exports land in `ZoteroExports/` and are refined into polished reference notes here.

## Linking

Bibliography notes are linked **from** notes in `ATLAS/`, `EFFORTS/`, and `Prabin's KB ~/` to provide source backing for ideas. Each bibliography note uses the format:

```
author - title (year)
```

> 📚 Good ideas deserve good sources. This folder is where intellectual debt is acknowledged.
