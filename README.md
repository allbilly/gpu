## 🎯 What This Is

This is Prabin's personal **Obsidian vault** — a living, evolving knowledge base built using the **ACE (Atlas, Calendar, Efforts)** organizational framework, extended with a personal bibliography, visual diagrams via Excalidraw, and academic references from Zotero.

Selected notes marked with `dg-publish: true` are automatically published to the live digital garden at [prabins.vercel.app](https://prabins.vercel.app) via the [Digital Garden Plugin](https://github.com/oleeskild/Obsidian-Digital-Garden) and the [pk_base](https://github.com/PrabinKuSabat/pk_base) site template.

## 📁 Vault Structure

```
NotesByPrabin/
├── ATLAS/            # Maps of Content (MOC) — index notes linking ideas together
├── Bibliography/     # Reference notes, summaries, and source annotations
├── CALENDER/         # Daily notes, journals, and time-based records
├── EFFORTS/          # Active projects, goals, and ongoing work
├── Excalidraw/       # Visual diagrams, mind maps, and sketches
├── Extras/           # Miscellaneous notes and temporary files
├── Kanban Boards/    # Task and project management boards
├── Prabin's KB ~/    # Core personal knowledge base — the main hub
├── ZoteroExports/    # Academic references exported from Zotero
└── webclips/         # Saved and annotated web articles
```

## 🔄 Publishing Workflow

Notes in this vault can be selectively published to the live site:

1. Add `dg-publish: true` to a note's YAML frontmatter
2. Open the **Digital Garden plugin** panel in Obsidian → click **Publish**
3. The plugin pushes the note to [PrabinKuSabat/pk_base](https://github.com/PrabinKuSabat/pk_base) via the GitHub API
4. Vercel detects the push, rebuilds, and the note is live at [prabins.vercel.app](https://prabins.vercel.app) ⚡

## 🛠️ Key Plugins Used

| Plugin | Purpose |
|--------|---------|
| [Digital Garden](https://github.com/oleeskild/Obsidian-Digital-Garden) | Publish selected notes to the live website |
| [Excalidraw](https://github.com/zsviczian/obsidian-excalidraw-plugin) | Create and embed visual whiteboard diagrams |
| [Zotero Integration](https://github.com/mgmeyers/obsidian-zotero-integration) | Import academic references directly from Zotero |
| [Kanban](https://github.com/mgmeyers/obsidian-kanban) | Visual drag-and-drop task management boards |
| [Dataview](https://github.com/blacksmithgu/obsidian-dataview) | Query notes as a dynamic database |
| [ExcaliBrain](https://github.com/zsviczian/excalibrain) | Interactive visual graph of note relationships |

## 🔒 Privacy Note

This vault is a **private personal knowledge base**. The majority of notes are not published publicly. Only notes with `dg-publish: true` in their frontmatter appear on the live site. Everything else stays local.
