# 🗂️ Extras

Miscellaneous files, temporary notes, and anything that doesn't yet have a clear home in the vault structure.

## What Lives Here

- **Experimental notes** — drafts and ideas being explored before being categorized
- **Imported files** — content imported from external sources pending review
- **Temporary notes** — short-lived notes that will be processed and moved
- **Plugin outputs** — auto-generated files from certain Obsidian plugins
- **One-off files** — notes that are useful once but don't belong in the main structure

## Maintenance

This folder should be reviewed periodically. Notes here are in one of two states:

1. **Waiting to be processed** — move to `ATLAS/`, `EFFORTS/`, `Bibliography/`, or `CALENDER/`
2. **Genuinely miscellaneous** — keep here or delete if no longer useful

> 🗂️ A clean Extras folder is a sign of a well-maintained vault. When in doubt, file it here first, process it later.
