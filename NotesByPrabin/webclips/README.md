# 🌐 webclips

Saved and annotated web articles, clipped directly from the browser into Obsidian. This folder is the **web reading inbox** of the vault.

## What Lives Here

- **Clipped articles** — full-text saves of web pages using a browser clipper extension
- **Annotated reads** — articles with personal highlights and inline commentary
- **Link archives** — important URLs saved with context before the page disappears
- **Thread captures** — saved social media threads, forum posts, or discussions

## Recommended Clipping Tools

| Tool | Description |
|------|-------------|
| [MarkDownload](https://github.com/deathau/markdownload) | Browser extension to clip any web page as clean Markdown |
| [Obsidian Web Clipper](https://obsidian.md/clipper) | Official Obsidian browser extension |
| [Omnivore](https://omnivore.app/) | Read-later app with Obsidian sync |

## Processing Webclips

Clipped articles are **raw captures** — they need processing:

1. Read and highlight the most important passages
2. Write a 2–3 sentence summary at the top
3. Extract key insights as links to notes in `ATLAS/` or `Prabin's KB ~/`
4. Tag appropriately for future retrieval

> 🌐 The web is vast. Webclips is where you anchor the parts of it worth keeping.
